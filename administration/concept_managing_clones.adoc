:allow-uri-read: 
:icons: font
:imagesdir: ../media/

Sie können Klone managen, indem Sie Klone erstellen (mit der Einstellung *Aktionen* oder eine Backup-Kopie als Quelle mounten), Klone löschen oder Klone aufhängen.
